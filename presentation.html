ЦЕЛИ И ЗАДАЧИ

Гипотезы

Что мы проверяем

H1: Гипотеза поверхностного выравнивания

Выбор инструмента — элемент стиля ответа, а не следствие недостатка знаний. Дообучение LoRA на небольшом объёме данных статистически значимо повысит Accuracy до >90%.

Опирается на: Zhou et al. (LIMA, 2023), Gudibande et al. (2023)

H2: Влияние качества данных

Качество обучающих данных (сбалансированный микс реальных + синтетических) оказывает большее влияние на результат, чем простое увеличение объёма выборки.

Если гипотезы верны: эффективная адаптация коммерческой LLM возможна с минимальными затратами на разметку данных.

Метод дообучения

LoRA — Low-Rank Adaptation

Input x
→
W₀ (frozen)
Замороженные веса
+
B × A (trainable)
rank r ≪ d
→
Output
W = W₀ + BA,   rank(BA) = r ≪ d

Ключевые свойства

  • Обучается <1% параметров
  • Сохраняет качество полного fine-tuning
  • Работает через API (без доступа к весам)
  • Идеально для коммерческих LLM
Вывод: LoRA позволяет адаптировать коммерческую модель YandexGPT, обучая лишь компактные матрицы обновлений через Yandex Cloud API.
